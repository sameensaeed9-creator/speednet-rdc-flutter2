3) STRIPE API KEYS
Create/get keys from:
Stripe Dashboard > Developers > API keys

Add publishable key in:
config.js
stripe.publishableKey = 'pk_live_...'

Add secret key in:
backend/api_keys.php
$STRIPE_SECRET_KEY = 'sk_live_...';

Never put Stripe secret key in JavaScript or public HTML.

4) WISE API
Wise is normally used with a Wise Business account.
Get API token from:
Wise Business > Settings > Developer tools / API tokens

Add token/profile in:
backend/api_keys.php
